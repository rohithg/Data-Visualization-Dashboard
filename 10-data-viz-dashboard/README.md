# Data Visualization Dashboard

Professional analytics dashboard with Chart.js and D3.js for interactive data visualization.

## Features

- **Multiple Chart Types**: Line, bar, doughnut, and custom D3 visualizations
- **Real-time Metrics**: Key performance indicators with trend indicators
- **Interactive Charts**: Hover effects and tooltips
- **Responsive Design**: Adapts to all screen sizes
- **Custom Visualizations**: D3.js for specialized charts
- **Dynamic Data**: Simulated real-time data updates

## Technologies

- Chart.js (canvas-based charts)
- D3.js (SVG visualizations)
- Responsive CSS Grid
- Vanilla JavaScript

## Installation

```bash
python3 -m http.server 8090
```

Visit http://localhost:8090

## Charts Included

1. **Line Chart**: Revenue over time with area fill
2. **Doughnut Chart**: User distribution
3. **Bar Chart**: Product categories
4. **D3 Bar Chart**: Traffic sources with custom styling
5. **Product List**: Top selling products

## Skills Demonstrated

1. **Data Visualization**: Multiple charting libraries
2. **Chart.js Mastery**: Configuration and customization
3. **D3.js**: SVG manipulation and custom visualizations
4. **Responsive Design**: Grid layouts and media queries
5. **Data Processing**: Aggregation and formatting
